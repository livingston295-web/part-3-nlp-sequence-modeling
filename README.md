# NLP and Sequence Modeling Mini Project

## Project Overview
This project performs customer support sentiment classification using NLP preprocessing, TF-IDF vectorization, and Logistic Regression.

## Dataset
Customer Support Text Classification Dataset

## Tasks Completed
- Dataset understanding
- Text preprocessing
- TF-IDF vectorization
- Baseline model
- Sequence model explanation
- Attention and transformer reflection

## Model Used
Logistic Regression with TF-IDF
